# ETF Rotation V4 真实数据复盘

## 结论

V4 第一轮没有达到替代 V2 的标准。

- V2 仍然是当前最优基线。
- V4 相比 V3 有改善，说明“自身历史状态确认”方向是对的。
- 但 V4 的最终净值、年化收益和最大回撤都仍然弱于 V2，说明短期确认层还不够稳。

## 真实回测结果

### V2 / V3 / V4 对比

| 版本 | 最终净值 | 年化收益 | 最大回撤 | 胜率 | 结论 |
|---|---:|---:|---:|---:|---|
| V2 | 38.2945 | 35.50% | -21.60% | - | 当前最优基线 |
| V3 | 13.7257 | 24.39% | -30.04% | - | 明显失败 |
| V4-A | 13.6607 | 24.97% | -28.76% | 66.30% | 比 V3 好，但不够 |
| V4-B | 18.5232 | 28.25% | -28.76% | 67.66% | 比 V3 好，但仍弱于 V2 |

说明：

- 本轮真实数据中，V4-A 和 V4-B 已经不再等价。
- V4-B 比 V4-A 更好，但两者都没有恢复到 V2 的防守纪律。

## 关键观察

### 1. V4-A / V4-B 方向有效，但强度不够

V4 用“自身历史状态”替代 V3 的横截面中位数，确实修掉了一部分问题。
但 252 日窗口、40% 分位、-2% 底线这组规则仍然过度防守，切掉了不少 V2 原本会继续持有的风险趋势。

### 2. 胜率不能单独看

V4 的胜率不低，但最大回撤仍明显劣于 V2。
这说明 V4 可能更“碎”，但没有抓住真正贡献净值的大段趋势。

### 3. 防守纪律没有保住

V4 的最大回撤仍然接近 V3，同时收益明显弱于 V2，说明它没有形成更好的风险收益平衡。
后续应优先看：

- `V2 防守 / V4 风险` 的日期
- `V2 风险 / V4 防守` 的日期
- `V4-A / V4-B` 的差异日

## 这轮已经确认的实现状态

- `qualified == false` 时，`rejection_reason` 不再为空。
- `V4-A` 和 `V4-B` 已经不是同一件事。
- `runner`、`backtest`、`strategy` 三层已经打通。
- 三组测试已通过。

## 下一步建议

1. 输出 V2 / V4 差异日归因表。
2. 单独统计 `V2 防守 / V4 风险` 的收益贡献和回撤贡献。
3. 再决定是否进入 V4.1，而不是直接调参数。

## 最终判断

V4 第一轮是有价值的，但它证明的是：

> “和自己历史比” 比 V3 的“和别人原始收益比”更合理，
> 但当前规则仍然没有恢复 V2 的防守纪律。
