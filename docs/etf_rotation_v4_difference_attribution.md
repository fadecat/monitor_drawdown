# ETF Rotation V4 差异日归因

## 结论先行

V4 的问题不是“选得更激进”，而是“防守切得更重”。

从真实数据看：

- V4-A 和 V4-B 都比 V3 好
- 但 V4 仍明显弱于 V2
- 主要差异来自 `V2 风险 / V4 防守`

这说明 V4 的短期确认偏严，导致不少 V2 会持有的风险资产被提前踢掉。

## 差异日总览

### V2 vs V3

- 同仓：2339 天
- `V2 防守 / V3 风险`：230 天
- `V2 风险 / V3 防守`：0 天
- 风险内重排：387 天

这说明 V3 的主要问题是把一部分防守日放回了风险资产，同时还改坏了风险池内部选择。

### V2 vs V4-A

- `V2 防守 / V4-A 风险`：0 天
- `V2 风险 / V4-A 防守`：897 天
- 风险内重排：688 天

### V2 vs V4-B

- 同仓：1459 天
- `V2 防守 / V4-B 风险`：0 天
- `V2 风险 / V4-B 防守`：775 天

这两组最关键：

> V4 没有在 V2 已经防守的时候更激进，反而是把大量 V2 持有的风险日改成了防守。

## 归因判断

V4 没有恢复 V2 的核心原因，不是 Top1 排名本身，而是短期确认层太保守，导致大量本来应该继续持有的风险日被切到了防守资产。

所以这轮差异日归因给出的判断是：

1. V4 的短期确认偏严，且偏保守。
2. V4-B 比 V4-A 更接近可用版本，但还不够。
3. 下一步不该先调参，而应先查清 `V2 风险 / V4 防守` 的收益贡献。

## 结论

V4 第一轮证明了一个方向：

> 和自己历史比，比和别人原始收益比更合理。

但它也证明了另一点：

> 当前短期确认没有保住 V2 的趋势持有质量。
