# ETF 轮动研究暂停与 V2 实盘计划

## 结论

当前阶段暂停继续研究，回到 V2 做实盘试运行。

V2 之后的 V3、V3.1、V4、V4.1 都已经完成了有价值的探索，但没有一个版本在全样本上明确超过 V2。继续在同一份数据上调整短期确认、止损阈值或新增风控规则，边际价值已经下降，过拟合风险正在上升。

所以当前主线应冻结为：

- V2 作为实盘基线版本。
- V3 / V3.1 / V4 / V4.1 全部作为研究分支保留，不进入主线。
- 不继续设计 V5。
- 后续研究必须等新数据或独立标的池验证。

## 为什么暂停研究

从 V1 到 V4.1 的路径已经说明，V2 是真正有效的结构升级，而 V2 之后的版本主要是在同一组历史样本上寻找局部改善。

研究路径如下：

| 版本 | 核心变化 | 结论 |
|---|---|---|
| V1 | 简单涨幅排名，风险资产和防守资产混排 | 结构不够清楚 |
| V2 | 趋势质量分数、短期确认、防守分离 | 当前主基线 |
| V3 | 用横截面中位数替代 `return_10d > 0` | 弱市保护明显变差 |
| V3.1 | 给 V3 补绝对底线 | 只能部分修复 |
| V4 | 用自身历史状态确认替代横截面比较 | 比 V3 合理，但没有恢复 V2 |
| V4.1 | 在 V2 上增加持仓峰值回撤止损 | 5% 档接近 V2，但改变持仓节奏 |

关键问题不是没有新想法，而是可用样本已经被反复观察。2025 年差异日、V4 的参数、V4.1 的止损阈值都已经进入样本内研究范围。继续微调容易把策略变成“解释过去”的系统，而不是可迁移的交易规则。

## 为什么回到 V2

V2 的优势不是某个参数，而是结构清楚：

- `score_25` 只负责排名。
- `return_10d > 0` 只负责短期确认。
- 511880 只在风险资产全不合格时接管。
- 防守资产不参与风险资产排名。

这个结构在当前回测中仍然是最稳的基线。

| 版本 | 最终净值 | 年化收益 | 最大回撤 | 主要判断 |
|---|---:|---:|---:|---|
| V2 | 38.2945 | 35.50% | -21.60% | 当前主基线 |
| V3 | 13.7257 | 24.39% | -30.04% | 明显失败 |
| V4-B | 18.5232 | 28.25% | -28.76% | 比 V3 好，但仍弱于 V2 |
| V4.1-5% | 37.8836 | 35.37% | -21.29% | 接近 V2，但止损触发 40 次 |

V4.1-5% 的结果接近 V2，但不能把它视为 V2 的简单增强版。它从 2014 到 2026 触发了 `40` 次止损，说明策略行为已经发生变化：一部分退出不再来自正常排名和防守切换，而是来自持仓中止损。

这件事需要未来新数据验证，不能因为当前样本中接近 V2 就直接替代。

## V2 实盘试运行要观察什么

实盘试运行的目标不是马上追求收益最大化，而是验证回测之外的问题。

重点观察四类问题。

### 1. 信号稳定性

需要记录每天的：

- 当前持仓
- 风险资产候选池
- Top1 标的
- `score_25`
- `return_10d`
- 是否切到 511880

重点看信号是否频繁在相邻标的之间来回切换，以及这种切换是否来自真实趋势变化。

### 2. 数据质量和延迟

回测默认数据完整、收盘价可用、信号计算口径一致。实盘中要单独观察：

- AkShare 或其他数据源是否延迟
- ETF 收盘价是否缺失或异常
- 除权、停牌、上市时间不足是否影响计算
- 信号日和执行日的数据是否严格按 T+1 处理

这些问题比继续微调参数更现实。

### 3. 执行偏差

V2 回测隐含了较理想的执行环境。实盘中要记录：

- 信号生成时间
- 实际下单时间
- 成交价格
- 与回测使用价格的偏差
- ETF 流动性和买卖价差

如果执行偏差长期明显，优先修执行流程，而不是改策略公式。

### 4. 心理和流程纪律

V2 的关键是规则统一。实盘试运行期间不要因为单日波动临时改规则。

需要提前确认：

- 信号是否按固定时间生成
- 是否允许人工跳过交易
- 如果数据异常，如何标记并暂停
- 是否保留完整日志，方便复盘

没有流程纪律，策略再简单也会在实盘中变形。

## V4.1 怎么处理

V4.1-5% 可以保留为观察版本，但不进入实盘主线。

保留它的目的不是继续调参，而是未来有新数据后回答一个问题：

> 持仓峰值回撤止损是否能稳定减少趋势转坏后的回撤，而不是在正常波动里频繁打断持仓？

在新数据到来前，不再测试：

- 4%、5%、6% 等更细止损网格
- 冷却期
- 最低持仓期
- ATR 止损
- RSI 过热过滤
- 复合风控打分

这些都可能让研究重新回到样本内优化。

## 什么时候重启研究

只有两种情况适合重启策略研究。

### 1. 新交易日积累足够多

等 2026 年下半年或更久的新交易日形成真正的新样本后，再检查：

- V2 是否继续稳定
- V4.1-5% 是否仍然改善回撤
- 止损触发次数和触发场景是否稳定
- 是否出现 V2 无法处理的新型风险场景

这类验证必须先冻结规则，再看新数据，不能边看边调。

### 2. 使用完全独立的新标的池

如果要验证规则泛化能力，可以换一批没有参与过调参的 ETF 池。

新标的池必须提前定义纳入标准，例如：

- 流动性
- 上市时间
- 跟踪误差
- 资产类别覆盖
- 是否允许行业 ETF
- 是否允许跨境和商品 ETF

不能根据回测表现挑标的，否则资产池本身也会变成过拟合来源。

## 当前行动清单

下一阶段只做 V2 实盘准备。

建议顺序：

1. 固化 V2 配置和资产池。
2. 输出每日 V2 信号日志。
3. 建立实际成交记录表。
4. 对比信号、执行价和回测假设。
5. 每周复盘一次执行偏差和数据质量，不改策略参数。

在 V2 实盘记录形成之前，不继续新增研究版本。

## 最终判断

V2 之后的研究不是白做。它确认了三件事：

1. V3 证明原始短期收益横截面比较会破坏弱市保护。
2. V4 证明自身历史比较更合理，但仍可能过度防守。
3. V4.1 证明持仓风控有研究价值，但当前样本不足以支持替代 V2。

所以当前最理性的选择不是继续找更复杂的规则，而是让 V2 进入实盘观察。

一句话：

> 研究先停，V2 先跑；新数据出现前，不再用旧样本拧新参数。
